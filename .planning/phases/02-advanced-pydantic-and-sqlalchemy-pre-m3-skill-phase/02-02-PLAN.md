```python
def at_least_one_field_provided(self) -> Self:
    fields_to_check = [
        'first_name', 'last_name', 'username', 'date_of_birth',
        'phone_number', 'address', 'occupation', 'emergency_contact',
        'medical_info', 'therapy_preferences', 'user_profile_data', 'terms_accepted'
    ]
    if all(getattr(self, f) is None for f in fields_to_check):
        raise ValueError("At least one profile field must be provided")
    return self
```

**TherapyPreferences** — add @field_validator:
```python
ALLOWED_COMMUNICATION_STYLES = ["supportive", "direct", "analytical", "casual", "formal"]

@field_validator("communication_style")
@classmethod
def validate_communication_style(cls, v: Optional[str]) -> Optional[str]:
    if v is not None and v not in cls.ALLOWED_COMMUNICATION_STYLES:
        raise ValueError(f"communication_style must be one of: {cls.ALLOWED_COMMUNICATION_STYLES}")
    return v
```
Note: Make `ALLOWED_COMMUNICATION_STYLES` a class variable (not inside the validator).
